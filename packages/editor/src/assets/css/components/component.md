COMPONENTS layer = “semantic objects”

This is the highest level.
This is where meaning appears.
Philosophy:
Components define what the user recognizes and interacts with
These are:
button
slider
panel
graph
toolbar
Example:

.button {
  background: var(--accent);
  border-radius: var(--radius);
}

.panel {
  background: var(--panel);
  border: 1px solid var(--border);
}

Now you're defining objects in your UI world.
